    aOS API Introduction

    Welcome to the aOS API for AriseBank! You can use our API to access aOS API endpoints, which can get information on various commands throughout our banking system.

    We have language bindings in Shell, Ruby, and Python! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    This API documentation is currently maintained by Jared Rice Sr (@JaredRiceSr) and Stanley Ford (@TheStormkrow) .

    Authentication

    To be authorized to access our API nodes, you must have developer keys. You can get your developer keys by creating a developer account at http://developer.arisebank.com.

    require 'aosarrow'

api = aOSArrow::APIClient.authorize!('meowmeowmeow')

    import aosarrow

api = aosarrow.authorize('meowmeowmeow')

    # With shell, you can just pass the correct header with each request
curl "api_endpoint_here"
  -H "aOSAPIKey: ariseapikey"
  -H "aOSSecretKey: arisesecretkey"

    Make sure to replace ariseapikey with your Arise Developer API key and arisesecretkey with your Arise Developer Secret Key

    aOS uses API keys to allow access to the API. You can register a new Arise Developer API / Secret keys at our developer portal.

    aOS expects for the API keys to be included in all API requests to the server in a header that looks like the following:

    aOSAPIKey: ariseapikey

    aOSSecretKey: arisesecretkey

    You must replace ariseapikey with your Arise Developer API key.
    You must replace arisesecretkey with your Arise Developer API key.

    Errors

    This error section is stored in a separate file in `includes/_errors.md`. Slate allows you to optionally separate out your docs into many files…just save them to the `includes` folder and add them to the top of your `index.md`’s frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    403 Forbidden – The kitten requested is hidden for administrators only
    404 Not Found – The specified kitten could not be found
    405 Method Not Allowed – You tried to access a kitten with an invalid method
    406 Not Acceptable – You requested a format that isn’t json
    410 Gone – The kitten requested has been removed from our servers
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slow down!
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
